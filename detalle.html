Título: Trabajo Práctico N° 1

Fecha de Inicio: 12 de Abril del 2025

Autores: Agustina Vasquez - Jonatan Fernando Calapeña Zarate - Juan Ignacio Arze - Esteban Sergio Vera - Sol Daiana Pizarro

Descripción

Este proyecto tiene como objetivo introducirnos en el desarrollo web mediante el uso de tecnologías básicas como HTML y CSS. Como estudiantes que recién comenzamos en este ámbito, buscamos comprender la estructura de una página web y aplicar estilos para mejorar su presentación visual.

A lo largo del proyecto, también se espera avanzar en el aprendizaje de herramientas más complejas como JavaScript y posiblemente React, con el fin de desarrollar interfaces más dinámicas e interactivas. Este trabajo representa un primer paso en nuestra formación como desarrolladores frontend.
