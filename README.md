## Descripcion del HTML
Las páginas HTML del proyecto fueron construidas siguiendo una estructura semántica clara y consistente, utilizando etiquetas como header, nav, main, section, article y footer para organizar el contenido de manera ordenada. Todas las vistas comparten un diseño común compuesto por una barra de navegación superior y un pie de página, lo que permite una navegación uniforme entre las distintas secciones del sitio.

Se desarrollaron cuatro páginas principales: una página de inicio con información general y estadísticas, una página de perfil con datos del usuario, una sección de proyectos que muestra diferentes trabajos en formato de tarjetas, y una página de detalle que amplía la información de cada proyecto. Además, se incorporaron hojas de estilo CSS externas para separar la presentación del contenido y mejorar la apariencia visual, junto con el uso de tipografías externas para lograr una interfaz más moderna y legible.

## Descripcion del CSS
En el CSS del proyecto se desarrolló un sistema de estilos modular y coherente que unifica la apariencia de todas las páginas. Se definieron variables globales en :root para gestionar colores, tipografías y estilos base, lo que permite mantener consistencia visual y facilitar futuras modificaciones.

Se implementó un diseño moderno utilizando técnicas como Flexbox y Grid para la distribución de elementos, logrando layouts adaptables tanto en la navegación, secciones de contenido y tarjetas de proyectos . Además, se incorporaron sombras, bordes suaves y gradientes para generar profundidad visual y jerarquía en los componentes.

El CSS también incluye interacciones visuales mediante estados :hover, :focus y :active, aportando dinamismo a botones, enlaces y tarjetas, mejorando la experiencia del usuario.

Finalmente, se mantuvo una separación clara entre estilos globales y específicos por página, organizando los estilos en múltiples archivos CSS según cada vista (inicio, perfil, proyectos y detalle), lo que mejora la escalabilidad y el mantenimiento del proyecto.
